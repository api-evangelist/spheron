# spheron

Spheron Network — decentralized GPU and cloud compute marketplace aggregating enterprise NVIDIA GPUs (H100, H200, B200, B300, A100, GH200, L40S, RTX PRO 6000/5090/4090) from certified Tier 3/4 data centers with per-minute billing, spot, reserved, and custom-cluster options.

## Developer Surfaces

- **API** — REST at `https://app.spheron.ai/api` (Bearer-token auth) — [docs](https://docs.spheron.ai/api-reference)
- **SDK** — [Protocol SDK (TypeScript)](https://github.com/spheron-core/protocol-sdk)
- **MCP Plugin** — [spheron-mcp-plugin](https://github.com/spheron-core/spheron-mcp-plugin)
- **SkyPilot** — [Spheron fork](https://github.com/spheron-core/skypilot)
- **Helm Charts** — [spheron-stack](https://github.com/spheron-core/spheron-stack)
- **ICL Genie** — [LLM-powered config generator](https://github.com/spheron-core/icl-genie-api)
- **Templates** — [awesome-spheron](https://github.com/spheron-core/awesome-spheron)
